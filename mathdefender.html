Math Defender

  • Nom : Math Defender
  • Genre : Serious Game
  • Thème : Scolaire

Infos

Math Defender est un Serious Game en cours de développement. Il est développé dans le cadre du projet OCTAVIA.
Son but est d'entraîner les élèves de niveau CE1 et CE2 aux calculs mentaux. A ce stade, les calculs sont générés de façon aléatoire. Prochainement, il sera possible de configurer différents paramètres (différents types de calculs, leçons, etc).
