Void Space

  • Nom : Void Space
  • Genre : Shoot'em up
  • Thème : Space Shooter

Contexte

Space Void est le premier jeu que j'ai développé avec Unity.
Space Void a été réalisé dans le cadre du Code Camp (ou Hackhaton interne à l'ETNA) Microsoft. L'objectif était de produire un jeu fonctionnel en trois jours avec un gameplay simple. Le défi fut à la fois de découvrir et d'apprendre les bases de du langage C# et de Unity 3D.
Space Void a été sélectionné pour être parmi les 10 meilleurs réalisations de ce Code Camp. Ainsi, j'ai pu présenter Space Void chez Microsoft à un jury de professionnels. Space Void a été classé 4ème sur les 10 projets concurrents.

Description

Space Void n'est pas un Shoot'em up classique. Les ennemies ne sont pas habituels
