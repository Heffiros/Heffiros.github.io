Ninja Run World

  • Nom : Ninja Run World
  • Genre : Aventure
  • Thème : Zombie Apocalypse

Infos

“Les zombies sortent de terre pour prendre le contrôle des grandes capitales. Incarnez un ninja, parcourez les niveaux et détruisez les portails d'où sortent les ennemis. Sauvez l'humanité de l'apocalypse zombie.”

Ninja Run World est le second jeu que j'ai réalisé avec Unity. Fort de mon expérience avec Void Space, j'ai voulu réalisé un jeu plus cartoonesque.
Grâce à de beaux assets graphiques, je me suis amusé à faire du level design. Il y a deux niveaux : le premier est un "environnement urbain", le second est celui de l'"Egypte antique". J'avais le projet de faire un niveau "Grande Muraille de Chine" et un autre "Ruines de temple Inca"
